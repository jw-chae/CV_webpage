Who Am I?

I am a Computer Vision researcher at Tsinghua University (Shenzhen International Graduate School), focusing on practical applications of deep learning in medical imaging and visual recognition systems.

My research interests include:

  • Computer Vision & Image Segmentation
  • Medical Image Analysis
  • Machine Learning & Deep Learning
  • Large Language Models in Healthcare
  • 3D Vision & Reconstruction
  • Object Detection & Tracking

Research Focus

My work focuses on developing practical computer vision solutions with applications in medical imaging and clinical decision support. I have published research on using machine learning and large language models for cancer staging, and I actively contribute to open-source projects related to visual segmentation, particularly the Segment Anything Model (SAM2).

Background

Currently pursuing research at Tsinghua University's Shenzhen International Graduate School at the Institute of Biopharmaceutical and Health Engineering. I maintain an active presence in the open-source community with 12+ repositories and growing contributions to computer vision research.

Research Projects

SAM2 GUI

Python • Computer Vision • GUI Development

A user-friendly graphical interface for the Segment Anything Model 2 (SAM2). Provides intuitive tools for real-time image segmentation with interactive annotation capabilities.

GitHub Repository

pLDDT Predictor

Machine Learning • Bioinformatics • Protein Structure

A machine learning-based model for predicting the confidence of protein structure predictions (pLDDT: predicted Local Distance Difference Test). Supports structural biology research by providing reliability assessments for protein models.

GitHub Repository

Memory-SAM

Deep Learning • Video Segmentation • Memory Optimization

A memory-efficient implementation of the Segment Anything Model. Optimized to perform high-quality video segmentation on limited hardware resources, making SAM more accessible for resource-constrained environments.

GitHub Repository

SAM2 Realtime

Real-time Processing • Jupyter Notebook • Interactive Demo

Real-time implementation of SAM2 for interactive segmentation tasks. Features a Jupyter Notebook-based demo that allows users to test and evaluate model performance in real-time scenarios.

GitHub Repository

GRID-AUGMENTED-VISION

Research Project • Object Tracking • Spatial Reference

Research project demonstrating that explicit spatial references (grid-based augmentation) can significantly enhance AI's ability to precisely locate and track objects without complex architectural modifications.

GitHub Repository

SJTU Projects

Academic Projects • Algorithms • Data Structures

Collection of academic projects from Shanghai Jiao Tong University (SJTU). Includes implementations of various algorithms and data structures.

GitHub Repository
