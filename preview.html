📤 GitHub에 배포하기

1단계: GitHub에 저장소 생성

  1. github.com/new 방문
  2. Repository name: CV_webpage
  3. Public 선택
  4. "Create repository" 클릭

2단계: 코드 푸시

cd /home/joongwon00/CV_webpage
git remote add origin https://github.com/jw-chae/CV_webpage.git
git commit -m "Initial commit: Personal portfolio website"
git push -u origin main

3단계: GitHub Pages 활성화

  1. GitHub 저장소 → Settings → Pages
  2. Source: main 브랜치 선택
  3. Save 클릭
  4. 몇 분 후 https://jw-chae.github.io/CV_webpage/에서 확인
